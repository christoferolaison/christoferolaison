# prettier-config

Shareable prettier config

## Installation

```bash
  yarn add @christoferolaison/prettier-config
```

```bash
  npm install @christoferolaison/prettier-config
```

## Usage

```javascript
// prettier.config.js

module.exports = require('@christoferolaison/prettier-config')
```
